# <a name="changes-to-external-user-access"></a>Zmeny prístupu externého používateľa

Od 23. marca 2018 nebudú mať externí používatelia prístup k obsahu, ktorý bol zdieľaný s každým, všetkými overenými používateľmi alebo všetkými používateľmi formulárov, a to ani v prípade, že boli pozvaní alebo pridaní do oprávnených skupín. Externí používatelia budú mať prístup len k obsahu zdieľanému s používateľom alebo so skupinami, do ktorých daný používateľ patrí. Ak chcete zachovať existujúce nastavenia, pozrite si téme: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
