- Ďalšie informácie o rozhraní Graph API v1.0 nájdete v téme [Microsoft Graph REST API v1.0.](https://docs.microsoft.com/graph/api/overview?toc=.%2Fref%2Ftoc.json&view=graph-rest-1.0)
